## 🎮 Як використовувати

### В грі:
1. Введіть ім'я та створіть/приєднайтеся до гри
2. Натисніть кнопку "🎲 Кинути кубик"
3. Відкрийте Developer Tools (F12) → Console
4. Побачите структуровані логи з результатами

### Панель розробника:
1. Натисніть кнопку 🛠️ в правому верхньому куті
2. Увімкніть/вимкніть логування
3. Очистіть консоль при потребі

### Ручне тестування:
1. Відкрийте Developer Tools (F12) → Console
2. Введіть: `window.testCashflowLogging()`
3. Натисніть Enter
4. Побачите серію тестових логів
