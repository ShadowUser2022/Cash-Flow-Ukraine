# Cash Flow Ukraine Clean — Roadmap

## Стан зараз (v0, staging)

- **Core-цикл**: `кубик → рух → подія → рішення` (pending deal/expense/salary/freelance/level-up). ✅
- **Профілі на старті** (Двірник / Поліцейська / Менеджер). ✅
- **Фінанси**:
  - **Підробіток**: на клітинці `payday`, невелика сума (≈ зарплата/40), інколи **“Спадок”** (кулдаун 50 ходів). ✅
  - **День зарплати**: раз на 10 ходів, показує формулу `зарплата + пасивний - витрати`, + **автопогашення боргу**. ✅
  - **Банк/кредит**: ліміт боргу 50 000 грн, у витратах є “відсотки банку”. ✅
  - **Витрати по категоріях**: оренда/їжа/комунальні/розваги/діти/відсотки банку. ✅
  - **Баланс / міс.** у дашборді: `зарплата + пасивний - витрати`. ✅
- **Дошка**:
  - Rat Race: 24 клітинки (витрат більше, payday менше). ✅
  - Fast Track: 16 клітинок. ✅
  - Мобільний viewport: вікно дошки **3 клітинки** (desktop 5). ✅
- **Мета гри**:
  - Перехід на Fast Track: `passiveIncome > expenses`. ✅
  - Перемога: на клітинці `dream`, якщо `cash + passiveIncome >= dream.cost`. ✅
  - **Ціни мрій збільшені ×5** (щоб гра не закінчувалась миттєво). ✅
- **Staging**:

## Що зроблено (ключові зміни)

- Об’єднаний фінансовий дашборд: **Мета зверху** + **Доходи** / **Витрати+банк** з розкладкою.
- Покращена читабельність карток угод (опис + окремий стовпець “ціна/дохід/витрати”).
- Баланс економіки: зменшено payday, збільшено витрати на дошці, рідкісний спадок.
- Автопогашення боргу в “день зарплати”.
- Мобільні правки: компактний хедер, “назад” як стрілка, 3 клітинки дошки, порядок блоків.
