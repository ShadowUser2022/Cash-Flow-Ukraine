## 🎮 Огляд проекту

**Cash Flow Ukraine** — це повноцінна мультиплеєрна онлайн-гра, заснована на настільній грі Роберта Кійосакі "Cashflow". Гра навчає фінансової грамотності через симуляцію реальних фінансових рішень.

### Ключові особливості:

- 🎭 **Мультиплеєр до 6 гравців** одночасно
- 🎥 **Відео чат через WebRTC** для живого спілкування
- 💬 **Реальний час чат** та сповіщення
- 🤝 **Система переговорів** для спільних інвестицій
- 💰 **Реалістична фінансова симуляція**
- 🎲 **Повна ігрова механіка** (кубик, угоди, переміщення)
- 🔄 **Real-time оновлення** через Socket.IO

### Базова концепція Cashflow

```
🐭 RAT RACE (Щуряча гонка) → 🚀 FAST TRACK (Швидка доріжка) → 🏆 ПЕРЕМОГА
```

#### Основні правила (НЕЗМІННІ):

1. **Два кола гри**: Rat Race (24 клітинки) + Fast Track (16 клітинок)
2. **Умова переходу**: `Passive Income ≥ Monthly Expenses`
3. **Умова перемоги**: `Fast Track + Cash ≥ Dream Cost`

## 🏗️ Архітектура системи

### Frontend (React/TypeScript)

- **React 18** з TypeScript
- **Zustand** для state management
- **Socket.IO клієнт** для real-time
- **WebRTC** для відео чату
- **CSS Modules** для стилізації

### Backend (Node.js/TypeScript)

- **Express.js** з TypeScript
- **Socket.IO** для real-time подій
- **In-memory storage** (MVP), планується MongoDB
- **REST API** для керування іграми

### Shared (Спільні типи)

- **TypeScript інтерфейси** для всіх сутностей
- **Ігрові константи** (професії, угоди, правила)
- **Socket.IO події** стандартизовані

### Milestone 2.1: Фінансова система (ЗАВЕРШЕНО - серпень 2025)

#### ✅ Централізований FinancialCalculator

- Єдиний сервіс для всіх фінансових розрахунків
- Розрахунок: готівка, доходи, витрати, cash flow
- Перевірка фінансової свободи
- Можливість купівлі активів

#### ✅ Система професій

- 4-6 професій з збалансованими параметрами
- Стартові умови: зарплата, витрати, борги, готівка
- Вибір професії при вході в гру

#### ✅ Система мрій

- 5 категорій мрій (подорожі, нерухомість, освіта, бізнес, хобі)
- Вартість мрій від 100k до 1M+
- Перевірка досягнення мрії на Fast Track

#### ✅ UI/UX покращення

- Універсальна система карток (9 варіантів)
- Оптимізація лоббі інтерфейсу
- Responsive дизайн для мобільних
- Accessibility WCAG 2.1

**Результат:** Повна фінансова механіка працює, UI готовий для користувачів.

### Milestone 2.2: Базова ігрова механіка (ЗАВЕРШЕНО - вересень 2025)

#### ✅ Рух по ігровому полю

- Кидок кубика (1-6)
- Переміщення гравця по Rat Race
- Проходження через PAYDAY
- Синхронізація позицій через Socket.IO

#### ✅ Система клітинок

- PAYDAY: отримання зарплати
- SMALL_DEAL: малі угоди (нерухомість, бізнес)
- BIG_DEAL: великі угоди з аукціонами
- DOODAD: витрати (одноразові/постійні)
- CHARITY: благодійність

#### ✅ Система карток

- Мал і угоди: гаражі, таксі, мобільні додатки
- Витрати (Doodads): ремонти, медичні витрати
- Ринкові події: зміна цін на активи

#### ✅ Перевірка умов

- Фінансова свобода: passive income ≥ expenses
- Перехід на Fast Track
- Перевірка перемоги

**Результат:** Основна механіка гри працює, гравці можуть грати базову версію.

## ⚠️ Ризики та міграція

### Технічні ризики:

- **WebRTC складність:** Peer-to-peer з'єднання можуть бути проблематичними
- **Socket.IO масштабування:** Потрібно оптимізувати для 6+ гравців
- **Database міграція:** Перехід з in-memory на MongoDB

### Бізнес-ризики:

- **Час розробки:** Fast Track реалізація може зайняти більше часу
- **Тестування:** Важко тестувати мультиплеєр сценарії
- **UX складність:** Переговори та торги потребують продуманого UI

### Міграція план:

1. **Березень 2026:** Завершення Fast Track
2. **Квітень 2026:** WebRTC інтеграція
3. **Травень 2026:** Database міграція
4. **Червень 2026:** Production deployment
5. **Липень 2026:** Beta launch
