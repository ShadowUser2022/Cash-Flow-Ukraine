1. Відкрити http://localhost:5173
2. Створити гру в DEV-MODE
3. Використати "Тестові ефекти" для перевірки:
   - Купівля угод
   - Сплата витрат
   - Благодійність
   - Ринкові дії
4. Перевірити, чи правильно оновлюється баланс в реальному часі
