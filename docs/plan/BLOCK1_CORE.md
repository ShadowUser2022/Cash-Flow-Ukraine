# Блок 1 — Core

Ціль: проста і стабільна механіка Core Solo, яку потім можна розширити до мультиплеєру.

## Інваріанти

- Rat Race має 24 клітинки. ✅
- Fast Track має 16 клітинок.✅
- Хід: `roll 1-6 -> move -> подія клітинки`. ✅
- Зарплата приходить раз на 10 ходів через інформаційну подію. ✅
- Перехід на швидкісну доріжку: `passiveIncome > expenses`.✅
- Банкрутство: щомісячні витрати стали більші за зарплату + пасивний дохід. ✅
- Перемога на Dream: `cash + passiveIncome >= dream.cost`.✅
- На старті гравець обирає одну мрію.

## Core Події V0

- `payday`: клітинка-нагадування, реальна зарплата раз на 10 ходів. ✅
- `deal`: купити за готівку/кредит або пропустити. ✅
- `expense`: оплатити повністю або взяти в кредит.✅
- `dream`: перевірити умову перемоги. ✅

## Перевірено

- Roll завжди змінює позицію в межах активної дошки.
- Зарплата рахується як `зарплата + пасивний дохід - витрати` раз на 10 ходів.
- Deal accept списує ціну угоди або бере нестачу в банку.
- Deal accept додає дохід активу і регулярні витрати активу.
- Deal skip не змінює фінанси.
- Expense accept зменшує cash.
- Expense credit додає щомісячний платіж до expenses.
- Fast Track transition не губить останній хід.
- Dream win спрацьовує тільки на Dream клітинці.

## Остання Перевірка

- **Дата:** 2026-04-25
- **Метод:** API прогін через локальний backend.
- **Результат:** Core v0 checklist пройдено.
- **Підтверджено:** deal accept/skip, expense pay/credit, зарплата раз на 10 ходів, позиції в межах дошки, Fast Track transition, Dream win.
