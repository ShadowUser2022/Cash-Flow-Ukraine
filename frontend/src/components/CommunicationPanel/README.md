# Communication Panel

Панель комунікації для гри CASHFLOW з трьома іконками в правому нижньому куті та невеликими віджетами внизу екрана.

## Компоненти

### CommunicationPanel
Основний компонент, що містить три кнопки:
- 🔊 **Звук** - відкриває віджет налаштувань звуку
- 📹 **Відео** - відкриває віджет відео зв'язку
- 💬 **Чат** - відкриває віджет чату гри

### AudioWidget
Невеликий віджет налаштувань звуку (280px x 200px) з:
- Ввімкнення/вимкнення головного звуку
- Регулювання гучності звуків гри
- Регулювання гучності музики
- Регулювання гучності ефектів

### VideoWidget
Невеликий віджет відео зв'язку (320px x 240px), що використовує існуючий VideoChat компонент.

### ChatWidget
Невеликий віджет чату (300px x 350px), що використовує існуючий Chat компонент.

## Поведінка

- **Клік на іконку** - відкриває відповідний віджет внизу екрана
- **Клік поза віджетами** - закриває всі віджети
- **Тільки один віджет** - може бути відкритий одночасно
- **Не перекриває гру** - віджети розташовані так, щоб не заважати ігровому процесу

## Використання

```tsx
import CommunicationPanel from '../CommunicationPanel';

// В GameBoard компоненті
<CommunicationPanel gameId={game?.id || ''} />
```

## Розташування

- **Панель іконок**: Фіксована в правому нижньому куті
- **Віджети**: З'являються над панеллю іконок (bottom: 80px)
- **Анімація**: Плавна поява знизу (slideUp)

## Стилі

Всі компоненти використовують єдиний дизайн з glassmorphism ефектами та адаптивним дизайном для мобільних пристроїв.
